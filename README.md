# PredatorWeb

Aplicación web Blazor Server con .NET 10 y DevExpress Components.

## Características

- **Menú horizontal superior** con opciones de archivo
- **Sistema de login** con TabPage
- **Componentes DevExpress** para una interfaz moderna y profesional
- **Tema Fluent** con colores personalizados

## Tecnologías

- .NET 10
- Blazor Server
- DevExpress Blazor Components
- C# 14.0

## Requisitos

- Visual Studio 2026 o superior
- .NET 10 SDK
- Licencia de DevExpress Blazor

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/TU_USUARIO/PredatorWeb.git
```

2. Abrir el archivo `PredatorWeb.slnx` en Visual Studio

3. Restaurar los paquetes NuGet

4. Ejecutar la aplicación (F5)

## Estructura del Proyecto

- `/Components/Pages/` - Páginas de la aplicación
- `/Components/Layout/` - Layouts y componentes de navegación
- `/wwwroot/` - Archivos estáticos (CSS, imágenes)
